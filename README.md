 I trained a **Machine Learning model using the Linear Regression algorithm** on the **Insurance dataset**. In this project, I performed **data preprocessing and feature engineering** to prepare the dataset for modeling.

 I explored the data through **data visualization** to understand relationships between features such as smoker status and medical charges. After that, I applied **train-test split** to divide the dataset into training and testing sets.

Then, I trained the **Linear Regression model** and evaluated its performance using **R-Squared Error and Adjusted R-Squared Error** to measure prediction accuracy.

This project helped me understand the **end-to-end workflow of a Machine Learning project**, including data preprocessing, feature engineering, model training, and model evaluation.
